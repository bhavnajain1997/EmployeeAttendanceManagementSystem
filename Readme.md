# Employee Attendance Management System

## 🧾 Overview

The **Employee Attendance Management System** is a web-based application designed to track and manage employee attendance. Employees can log their attendance and view records, while admins can manage all attendance data.  

**Note:** This project is **not fully responsive** yet, so it is best viewed on desktop screens.

---

## 🛠️ Features

✔️ Employee login and profile  
✔️ Mark daily attendance  
✔️ View attendance reports  
✔️ Admin panel for managing employees and attendance  

*(Future improvement: Make the UI fully responsive for mobile and tablet screens)*
